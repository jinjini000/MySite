이 웹사이트는 HTML, CSS, 그리고 **JavaScript**를 사용하여 제작되었으며, 특히 사용자 경험(UX)을 향상시키기 위해 JavaScript로 **동적인 기능**들을 구현했습니다.

---

### ✨ 주요 특징 및 JavaScript 구현 기능

이 웹사이트의 핵심적인 기능과 사용자 인터페이스 개선은 `script.js` 파일에 포함된 JavaScript 코드를 통해 이루어졌습니다.

#### 1. 시간대별 환영 메시지 설정 ☀️/🌙

사용자가 웹사이트에 접속한 **현재 시간에 따라** `home` 섹션의 인사말이 자동으로 변경됩니다. 이는 사용자에게 개인화된 느낌을 제공합니다.

* **구현 원리:** `setGreetingMessage` 함수에서 `new Date().getHours()`를 사용하여 현재 시각을 얻고, 이 시각에 따라 다른 인사말(`좋은 아침입니다!`, `좋은 오후입니다!`, `좋은 저녁입니다!`, `좋은 새벽입니다!`)을 선택하여 `#home__title` 요소에 추가합니다.

#### 2. 포트폴리오 카테고리 필터링 및 애니메이션 🖼️

`work` 섹션에서 카테고리 버튼을 클릭하면 해당 카테고리(All, Front-end, Mobile, Back-end)의 프로젝트만 표시되며, 이 과정에 **부드러운 전환 효과**를 적용했습니다.

* **카테고리 선택 하이라이트:** 클릭된 카테고리 버튼에만 `category--selected` 클래스를 적용하여 시각적인 피드백을 제공합니다.
* **프로젝트 표시/숨기기:** 각 카테고리 버튼의 `onclick` 이벤트에서 해당 카테고리의 프로젝트 요소(`project frontend`, `project backend`, `project mobile`)의 `display` 속성을 `block` 또는 `none`으로 설정하여 필터링합니다.
* **전환 애니메이션 (`itemTransition`):** 프로젝트가 나타날 때 (`display: 'block'` 일 때), `itemTransition` 함수를 호출하여 프로젝트 항목이 **위에서 아래로 내려오면서 투명도가 증가하고 크기가 커지는** 부드러운 전환 효과(transition, transform, opacity, scale)를 적용합니다.

#### 3. 네비게이션 메뉴 하이라이트 🧭

상단 `header`의 네비게이션 메뉴 항목을 클릭하면, 해당 항목에만 **`active` 클래스**를 적용하여 현재 사용자가 어느 섹션으로 이동했는지 시각적으로 알려줍니다.

* **구현 원리:** 각 메뉴 항목(`menuHome`, `menuAbout`, `menuSkills`, 등)의 `onclick` 이벤트 리스너에서 클릭된 항목에 `active` 클래스를 추가하고, 나머지 항목에서는 `active` 클래스를 제거하여 활성화 상태를 관리합니다.

### 🛠️ 사용 기술

* **HTML5:** 웹 페이지의 구조를 담당합니다.
* **CSS3:** 스타일링 및 레이아웃을 담당합니다.
* **JavaScript (ES6+):** 동적인 기능 및 사용자 인터페이스 상호작용을 구현합니다.
* **Font Awesome:** 아이콘 사용을 위해 사용되었습니다.
